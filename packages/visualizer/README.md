- üöÄ **Fast Performance**: Built with React and Bun.

## How to use Web Visualizer

1. Run the visualizer locally:

   ```bash
   bun visualizer:dev
   ```

2. Open your browser at `http://localhost:5173`.
3. Drag and drop your `.csv` log file onto the dashboard.
